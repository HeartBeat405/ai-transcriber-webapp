# AI Transcriber Web App

Modern AI-powered speech-to-text web application built using Flask and AssemblyAI API.

## Features
- Audio Upload
- AI Speech-to-Text
- Animated Progress Bar
- Download Transcript
- Responsive Modern UI

## Tech Stack
- Python
- Flask
- AssemblyAI API
- HTML
- CSS
- JavaScript

## Setup

```bash
pip install -r requirements.txt
python app.py
